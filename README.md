# The Hertie Times

A student data journalism showcase built for **GRAD-E1493: Data Journalism** at the Hertie School, Berlin (Spring 2026), taught by Prof. Simon Munzert.

Live site: <https://data-journalism-26.github.io/>

## What it is

A static news-magazine-style website that displays student data journalism projects. Projects span investigative data work, interactive visualizations, and explanatory journalism produced over the course of the semester.

## Tech

Plain HTML, CSS, and JavaScript — no frameworks, no build step. Project metadata lives in `projects.json` and is rendered client-side. Deployed via GitHub Pages.

## Structure

```
index.html       page shell
style.css        all styles
app.js           client-side rendering from projects.json
projects.json    project metadata
thumbnails/      author portrait photos
```
